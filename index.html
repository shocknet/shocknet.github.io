Welcome to Shock

Shock is a layer 3 implentation of an encrypted identity and messaging protocol exchanged over Bitcoin-Lightning. "ShockApp" is a wallet software who's goal is to such features in a social context that is friendly to novice users.

The purpose of this project is not only to create a user-friendly Lightning wallet, but to do so in a manner that drives adoption to Bitcoin as a utility in day-to-day interactions.
